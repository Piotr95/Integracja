#  HRM Human resource management system




## Steps to Setup

 # I. Install Lombok plugin in your IDE (if needed)
 # II Run Applications from IDE or terminal/cmd by : 
```
mvn spring-boot:run
```
 Order of Aplpications running :
 ```
 MicroservicesIntegration -> other applications -> HRM
  ```
  # III For  API  usage  examples  go to :
 http://localhost:(currentAplicationPort)/swagger-ui.html
 

Useful links : 

```
https://swagger.io/

https://www.baeldung.com/swagger-2-documentation-for-spring-rest-api
```

# Enjoy :)
